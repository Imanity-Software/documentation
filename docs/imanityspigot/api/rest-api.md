# Rest API

## Download

HTTP Method: `GET`

License Type: `Enterprise`

URL: `https://imanity.dev/raisensu/download/ispigot3`

Parameters:
- `version`: The version of ImanitySpigot3. `(latest/latest/<yyyy>.<mm>.<release>b<build>[-LTS])`
  
Response Data:
- JAR data
